# ✨ Markdown Studio

A modern, fast, and production-grade Markdown workspace for developers and creators.

🔗 **Live Demo:** https://markdown-studio-code.vercel.app/

---

## 🚀 Features

* ⚡ **Live Preview** – See your Markdown rendered instantly
* 🧠 **VS Code Feel** – Clean, developer-friendly interface
* 📄 **Export Options** – Export to `.md`, `.html`, `.pdf`
* 🎯 **Distraction-Free Writing** – Focus on content only
* 📱 **Responsive Design** – Works on desktop and mobile

---
